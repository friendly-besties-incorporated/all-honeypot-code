Passphrase Security Questions

Your answers to the Passphrase Security Questions will provide information that will be used to verify your identity if you forget your passphrase.
If you haven't already, you may consider also setting up a verified phone which can be used as an alternate method of verifying your identity. You can do this by visiting your Directory Profile page.
If you have an Associate Account Email, you will be unable to set Security Questions for that Associate Account.

Instructions: Select a question from each set. Please supply an answer for all three questions that you will remember in years to come.
